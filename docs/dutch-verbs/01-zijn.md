## Conjugation

### Present

|                 | PRESENT TENSE | PRESENT PERFECT TENSE |
| :-------------: | ------------: | --------------------: |
|       Ik        |       **ben** |       **ben geweest** |
|  Jij / Je / U   |      **bent** |      **bent geweest** |
| Hij / Zij / Het |        **is** |        **is geweest** |
|    Wij / We     |      **zijn** |      **zijn geweest** |
|     Jullie      |      **zijn** |      **zijn geweest** |
|       Zij       |      **zijn** |      **zijn geweest** |
